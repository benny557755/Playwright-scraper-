# Multi-Page Product Scraper (Playwright)

## 🚀 Description
A Python web scraper that extracts product name and price from multiple pages using Playwright.

## ⚙️ Features
- Multi-page pagination scraping
- Product name & price extraction
- Anti-bot basic handling
- CSV & JSON export

## 🛠 Tech Stack
- Python
- Playwright
- Pandas

## 🚀 How to Run
```bash
pip install -r requirements.txt
playwright install
python main.py
