# Multi-Page Product Scraper (Python + Playwright)

A Python-based web scraping tool using Playwright to extract product data (name, price) from multi-page e-commerce websites and export results to CSV and JSON.

## Features
- Multi-page pagination scraping
- Extract product name and price
- Anti-bot resource optimization
- Export data to CSV and JSON
- Configurable scraping limits

## Tech Stack
- Python
- Playwright
- Pandas

## Installation

```bash
pip install -r requirements.txt
playwright install
